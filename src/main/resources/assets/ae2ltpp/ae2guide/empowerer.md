# Empowerer

**Packaged Core:** <ItemLink id="ae2ltpp:aa_empowerer_packaged_core" />
**Mod:** Actually Additions

Automates Empowering recipes (one base item + four modifier items → one empowered item).
Place the provider against the Empowerer and encode the recipe as a pattern.

## Structure

* The 4 **Display Stands** must sit on the **third block** north, south, east, and west of the
  Empowerer, on the same level. In other words, leave **2 empty blocks** between the Empowerer and
  each Display Stand — the standard Empowerer layout.
* The Display Stands need **power (RF)**, just like normal Empowering.
* Before starting, the Empowerer and all 4 Display Stand item slots must be empty.

The provider places the base item on the Empowerer and the four modifiers on the stands, then lets the
machine finish and pulls the result back.
