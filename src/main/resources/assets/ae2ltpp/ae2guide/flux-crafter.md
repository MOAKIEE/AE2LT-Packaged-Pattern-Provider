# Flux Crafter

**Packaged Core:** <ItemLink id="ae2ltpp:ec_flux_crafter_packaged_core" />
**Mod:** Extended Crafting

Automates the Flux Crafter — the same 3×3 layout as the Ender Crafter, but progress is powered by **FE**
instead of time. Place the provider against the Flux Crafter and encode the recipe as a pattern.

## Requirements

* Place **Flux Alternator** blocks and supply FE according to the Flux Crafter's normal rules.
* The craft advances as power is drawn from the alternators; if power is low it simply takes longer —
  items won't be lost while it waits.
* Before starting, the 3×3 grid and output slot must be empty.
