# Mana Pool

**Packaged Core:** <ItemLink id="ae2ltpp:botania_mana_pool_packaged_core" />
**Mod:** Botania

Automates Mana Pool infusion (one input item → one output item). Place the provider against the Mana
Pool and encode the recipe as a pattern; the result is returned right away.

## ⚠ Mana & catalyst block

* The pool must hold enough **mana** for each craft — keep it fed by your mana spreaders.
* Some recipes require a **catalyst block under the pool** (e.g. Alchemy Catalyst or Conjuration Catalyst).
  Place the correct catalyst block, exactly as the recipe needs — the pool picks the matching recipe based
  on the block beneath it.
