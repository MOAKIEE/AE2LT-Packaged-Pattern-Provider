# 花药台

**所需封包核心：** <ItemLink id="ae2ltpp:botania_petal_apothecary_packaged_core" />
**来源模组：** Botania（植物魔法）

自动完成花药台配方（花瓣 + 一颗种子 → 神秘花朵）。把供应器贴着花药台放置，将配方做成样板。
没有合成延迟，产物会立刻返回。

## ⚠ 每次合成耗水

每次合成会像手动那样消耗花药台中的**一份水**。你可以让台子里一直有水，或在样板输入中加入一个**水源**
（例如一桶水），让供应器自行补给。种子／试剂属于普通配方材料，会被消耗。
