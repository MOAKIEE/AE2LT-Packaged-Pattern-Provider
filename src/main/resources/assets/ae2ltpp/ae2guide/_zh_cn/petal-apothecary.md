# 花药台

**所需封包核心：** <ItemLink id="ae2ltpp:botania_petal_apothecary_packaged_core" />
**来源模组：** Botania（植物魔法）

自动完成花药台配方（花瓣 + 一颗种子 → 神秘花朵）。把供应器贴着花药台放置，将配方做成样板。
没有合成延迟，产物会立刻返回。

## ⚠ 每次合成耗水

每次合成需要**一份水**。供水优先级为：样板输入中的水桶（返还空桶）、样板输入中的水流体（每次 1000 mB）、
花药台已有的水。种子／试剂属于普通配方材料，会被消耗。
