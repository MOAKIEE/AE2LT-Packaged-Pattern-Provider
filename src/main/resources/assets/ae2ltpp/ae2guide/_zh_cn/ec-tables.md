# 工作台（基础 / 高级 / 精英 / 终极）

**所需封包核心：** <ItemLink id="ae2ltpp:ec_basic_table_packaged_core" />
<ItemLink id="ae2ltpp:ec_advanced_table_packaged_core" />
<ItemLink id="ae2ltpp:ec_elite_table_packaged_core" />
<ItemLink id="ae2ltpp:ec_ultimate_table_packaged_core" />
**来源模组：** Extended Crafting（扩展合成）

自动完成扩展合成的四种工作台（基础 3×3、高级 5×5、精英 7×7、终极 9×9）。把供应器贴着工作台放置，
将工作台配方做成样板；产物（以及任何容器／剩余物，例如空桶）会即时返回。

## ⚠ 高阶封包核心向下兼容

四个封包核心构成一条链：**高阶封包核心会解锁所有低阶工作台**。终极工作台封包核心可驱动终极／精英／高级／基础，
精英工作台封包核心可驱动精英／高级／基础，依此类推。你只需准备最高那一档工作台对应的封包核心即可。

## 注意事项

* 工作台的合成格必须**空置**——不要在里面留物品，也不要同时用手动合成。
* 合成的副产物（空桶之类）会随产物一起返回。
