# 聚合合成

**所需封包核心：** <ItemLink id="ae2ltpp:de_fusion_packaged_core" />
**来源模组：** Draconic Evolution（龙之进化）

自动完成聚合合成。把供应器贴着**聚合合成核心**放置。供应器会自动识别周围的**聚合合成注入器**，
将催化物放入核心、每个材料放入一个注入器，启动合成，完成后取回产物。

## 要求

* 按正常方式搭好多方块：一个合成核心 + 足量**注入器**环绕，并接好能量。
* 注入器**等级**必须达到配方要求（魔龙 / 龙 / 混沌）。等级不够的注入器会被跳过，请确保达标等级的注入器数量足够。
* 开始前核心的催化物槽和本次要使用的注入器必须为空。
* 催化物属于普通配方材料，会被消耗。
