# 符文祭坛

**所需封包核心：** <ItemLink id="ae2ltpp:botania_runic_altar_packaged_core" />
**来源模组：** Botania（植物魔法）

自动完成符文祭坛配方。把供应器贴着符文祭坛放置，将配方做成样板。供应器会把材料送入祭坛；
当它积攒到足够魔力时产出产物并返回（你无需再用法杖右键）。

## ⚠ 魔力与「会回收的催化符文」

* 祭坛需要来自魔力散发器的**魔力**——请保持供给。
* 有催化符文的配方：样板输入写消耗材料 + 催化符文；样板输出写主产物 + 每个会退回的催化符文。
* 如果输出只写主产物，AE2 不会把退回符文计入这次合成。
