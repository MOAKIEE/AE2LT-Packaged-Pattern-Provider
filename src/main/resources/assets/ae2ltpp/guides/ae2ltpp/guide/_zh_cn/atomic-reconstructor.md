# 原子再构机

**所需封包核心：** <ItemLink id="ae2ltpp:aa_reconstructor_packaged_core" />
**来源模组：** Actually Additions（实用拓展）

自动完成原子再构机的激光转化（一个输入物 → 一个输出物）。把供应器贴着原子再构机放置，
将转化做成样板，产物会即时返回——不会真的在地上掉落物品。

## 注意事项

* 再构机仍然**需要能量（RF/CF）**。每次转化都会像手动那样消耗电力；电量不足时合成会暂停，等到有电再继续。
* **只有默认的「转化透镜」有效**。其它特殊透镜（挖掘、死亡、染色……）会被忽略，因为它们的附加效果无法安全自动化。
