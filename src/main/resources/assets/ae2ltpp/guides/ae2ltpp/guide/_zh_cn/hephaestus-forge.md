# 赫菲斯托斯锻炉

**所需封包核心：** <ItemLink id="ae2ltpp:fa_hephaestus_forge_packaged_core" />
**来源模组：** Forbidden Arcanus（禁忌与奥秘）

自动完成赫菲斯托斯锻炉的仪式。把供应器贴着锻炉放置。适配器会把主材料放入锻炉、其余材料放到周围基座上，
随后启动仪式，完成后取回产物（仪式有固定时长，并伴随常见的法阵特效）。

## 结构

* 锻炉约 4 格范围内最多 **8 个基座**（即锻炉的常规布置）。

## ⚠ 必须自行备足「精质」

仪式会消耗**精质**（灵气 / 灵魂 / 血液 / 经验）。本适配器**不会补给精质**——它只消耗锻炉里已有的部分。
请通过 Forbidden Arcanus 自身的方式（例如其精质拾取机制）为锻炉填充精质，否则仪式会一直等待，直到精质足够。
