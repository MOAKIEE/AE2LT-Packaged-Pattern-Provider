# Clibano (Combustion Chamber)

**Adapter card:** <ItemLink id="ae2ltpp:fa_clibano_adapter" />
**Mod:** Forbidden Arcanus

Automates Clibano smelting. Build the 3×3×3 Clibano and place the provider against the **Clibano Core**.
The adapter inserts the input and pulls out the result; the Clibano runs on its own burn time (~200–400
ticks per item).

## ⚠ You must keep Fuel & Souls stocked

The Clibano needs **Fuel** and **Souls** (and optionally an **Enhancer**) to run. These are **not** part
of the recipe pattern, so the adapter does not supply them — keep them topped up yourself.

If the **fire type** doesn't match the recipe (wrong fuel/soul), the input just waits in the Clibano
until the correct fire type is available — exactly what you'd see crafting by hand.
