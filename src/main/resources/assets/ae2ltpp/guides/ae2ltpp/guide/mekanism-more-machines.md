# Mekanism: More Machines — Large Machines

**Adapter card:** <ItemLink id="ae2ltpp:mekmm_adapter" />
**Mod:** Mekanism: More Machines

A single card drives **all** of the mod's large multiblock machines. Build the machine, place the
provider against its **controller**, install this one card, and encode the recipe as a pattern — the
adapter detects which machine it's facing automatically.

## Covered machines

* Large Chemical Infuser
* Large Electrolytic Separator
* Large Antiprotonic Nucleosynthesizer
* Large Rotary Condensentrator
* Large Solar Neutron Activator
* Large Pigment Mixer

## Notes

* The machine still needs its normal **energy**.
* Item, fluid, and Mekanism chemical inputs in the pattern are routed to the matching machine ports automatically.
* One card is enough no matter which of the above machines sits in front of the provider.
