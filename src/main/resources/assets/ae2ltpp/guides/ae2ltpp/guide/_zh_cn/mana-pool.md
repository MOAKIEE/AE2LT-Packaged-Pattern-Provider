# 魔力池

**所需适配卡：** <ItemLink id="ae2ltpp:botania_mana_pool_adapter" />
**来源模组：** Botania（植物魔法）

自动完成魔力池的魔力注入（一个输入物 → 一个输出物）。把供应器贴着魔力池放置，将配方做成样板；产物会立刻返回。

## ⚠ 魔力与催化方块

* 每次合成池中都要有足够的**魔力**——用魔力散发器持续为它供能。
* 部分配方需要在**池子下方放置催化方块**（例如 炼金催化方块 或 咒愿催化方块）。请按配方要求放上正确的催化方块——
  魔力池会根据下方方块选择对应的配方。
