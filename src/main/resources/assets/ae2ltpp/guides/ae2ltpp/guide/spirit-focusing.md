# Spirit Focusing (Spirit Crucible)

**Adapter card:** <ItemLink id="ae2ltpp:malum_spirit_focusing_adapter" />
**Mod:** Malum

Automates Spirit Crucible focusing. Place the provider against the **Spirit Crucible** and encode the
recipe as a pattern. Each craft, the adapter only inserts the recipe's **spirit shards**, and the result
is returned when focusing finishes.

## ⚠ Install the catalyst first

The crucible's **catalyst is reusable and must already be installed** in the crucible by you. The adapter
does **not** put the catalyst in — it only feeds the spirit shards. Set up the crucible with its catalyst
once, and it keeps working for every craft.
