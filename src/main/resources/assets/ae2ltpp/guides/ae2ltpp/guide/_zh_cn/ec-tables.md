# 工作台（基础 / 高级 / 精英 / 终极）

**所需适配卡：** <ItemLink id="ae2ltpp:ec_basic_table_adapter" />
<ItemLink id="ae2ltpp:ec_advanced_table_adapter" />
<ItemLink id="ae2ltpp:ec_elite_table_adapter" />
<ItemLink id="ae2ltpp:ec_ultimate_table_adapter" />
**来源模组：** Extended Crafting（扩展合成）

自动完成扩展合成的四种工作台（基础 3×3、高级 5×5、精英 7×7、终极 9×9）。把供应器贴着工作台放置，
将工作台配方做成样板；产物（以及任何容器／剩余物，例如空桶）会即时返回。

## ⚠ 高级卡向下兼容

四张卡构成一条链：**高级别的卡同时解锁所有较低级别**。所以一张**终极工作台**卡可驱动全部四种工作台，
精英卡可驱动 精英／高级／基础，依此类推。你只需准备**最高那一档**工作台对应的卡即可。

## 注意事项

* 工作台的合成格必须**空置**——不要在里面留物品，也不要同时用手动合成。
* 合成的副产物（空桶之类）会随产物一起返回。
